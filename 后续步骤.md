# EZProduct - 后续安装步骤

## ✅ 已完成
- [x] 第一步：进入项目目录
- [x] 第二步：安装依赖包（npm install）

## 📋 接下来需要执行的命令

### 第三步：生成Prisma数据库客户端（必须先执行）
```bash
npm run db:generate
```

### 第四步：创建数据库表
```bash
npm run db:migrate
```
执行时会提示输入迁移名称，直接按回车使用默认名称即可。

### 第五步：创建环境变量文件
```bash
cp .env.example .env
```

### 第六步：编辑.env文件
使用文本编辑器打开 `.env` 文件，填入API密钥：

```env
# Shopify App Configuration
SHOPIFY_API_KEY=你的Shopify_API_Key
SHOPIFY_API_SECRET=你的Shopify_API_Secret
SCOPES=write_products,read_products,write_product_listings,read_product_listings
SHOPIFY_APP_URL=https://你的应用URL

# Database
DATABASE_URL="file:./dev.db"

# AI API Configuration
GEMINI_API_KEY=你的Gemini_API_Key

# App Environment
NODE_ENV=development
```

### 第七步：启动开发服务器
```bash
npm run dev
```

## 🚀 一键复制（从第三步开始）

```bash
# 第三步：生成Prisma客户端
npm run db:generate

# 第四步：创建数据库
npm run db:migrate

# 第五步：创建.env文件
cp .env.example .env

# 提示
echo "✅ 请编辑 .env 文件填入API密钥，然后运行: npm run dev"
```

## ⚠️ 注意事项

1. **必须先执行第三步**：`npm run db:generate` 必须在 `npm run db:migrate` 之前执行
2. **迁移名称**：执行 `npm run db:migrate` 时，如果提示输入迁移名称，直接按回车使用默认名称
3. **API密钥**：必须填入 `.env` 文件后才能启动应用

## 🔍 验证步骤

每步执行后可以验证：

```bash
# 验证第三步：检查Prisma客户端是否生成
ls node_modules/.prisma/client 2>/dev/null && echo "✅ Prisma客户端已生成" || echo "❌ 未生成"

# 验证第四步：检查数据库文件是否创建
ls prisma/dev.db 2>/dev/null && echo "✅ 数据库已创建" || echo "❌ 数据库未创建"

# 验证第五步：检查.env文件是否存在
ls .env 2>/dev/null && echo "✅ .env文件已创建" || echo "❌ .env文件未创建"
```
