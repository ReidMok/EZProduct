我已经创建了认证路由文件 `app/routes/auth.$.tsx`，用于处理所有认证相关的路由。

### 路由文件说明

`auth.$.tsx` 是一个 Remix 的 catch-all 路由，会匹配所有 `/auth/*` 路径，包括：
- `/auth/callback` - OAuth 回调
- `/auth/login` - 登录页面
- `/auth/shopify/callback` - Shopify OAuth 回调
- `/api/auth/callback` - API 认证回调
