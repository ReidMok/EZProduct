## 🎯 启动应用

### 在终端中执行：

```bash
cd /Users/Zhuanz/cursor/resinmemory_product/ezproduct
npm run dev
```

## 📋 启动后会发生什么

1. **Shopify CLI 启动**
   - 会显示 "Starting local development server..."

2. **自动创建网络隧道**
   - Shopify CLI 会自动创建 Cloudflare Tunnel
   - 会显示一个临时的公网URL（例如：`https://xxx.tunnel.cloudflare.com`）

3. **打开浏览器**
   - 自动打开浏览器
   - 显示应用安装页面

4. **选择开发商店**
   - 选择你的Shopify开发商店
   - 点击 "Install" 安装应用

5. **应用启动**
   - 应用会在你的Shopify后台中打开
   - 可以看到 "EZProduct - AI Product Generator" 界面

## 🎨 测试应用

应用启动后：

1. **进入应用界面**
   - 在Shopify后台左侧菜单找到你的应用
   - 或直接访问应用URL

2. **测试产品生成**
   - 在输入框中输入产品关键词（例如："Three Divers Resin Night Light"）
   - 点击 "Generate & Sync Product"
   - 等待AI生成产品
   - 检查产品是否成功创建在你的Shopify店铺中

## ⚠️ 如果遇到问题

### 问题1：端口被占用
```bash
# 检查端口3000是否被占用
lsof -i :3000

# 如果被占用，可以修改端口或关闭占用端口的程序
```

### 问题2：Shopify CLI未安装
```bash
# 安装Shopify CLI
npm install -g @shopify/cli @shopify/theme
```

### 问题3：API密钥错误
- 检查 `.env` 文件中的API密钥是否正确
- 确保没有多余的空格或引号
- 重新从Shopify Partners复制API密钥

### 问题4：数据库连接错误
```bash
# 检查数据库文件是否存在
ls -la prisma/dev.db

# 如果不存在，重新运行迁移
npm run db:migrate
```
